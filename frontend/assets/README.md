# Assets Folder

## Required Images

### 1. Favicon
- **File:** `favicon.ico` or `favicon.png`
- **Size:** 32x32 or 64x64 pixels
- **Format:** ICO, PNG, or SVG

### 2. Background Image (Optional)
- **File:** `background.jpg` or `background.png`
- **Recommended Size:** 1920x1080 or higher
- **Format:** JPG, PNG, or WebP
- **Usage:** Home screen gradient overlay

### 3. App Icon (Electron)
- **File:** `icon.png`
- **Size:** 512x512 pixels (will be auto-scaled)
- **Format:** PNG with transparency
- **Usage:** Desktop app icon, taskbar icon

### 4. Logo
- **File:** `logo.png` or `logo.svg`
- **Size:** 256x256 pixels
- **Format:** PNG or SVG
- **Usage:** App header, about page

## How to Add Your Images

1. Place your images in this folder
2. The app will automatically detect and use them
3. If images are missing, app will use default colors/emojis

## Quick Start (No Design Skills?)

Use these free tools to create icons:
- **Favicon Generator:** https://favicon.io/
- **Icon Generator:** https://www.appicon.co/
- **AI Icon Generator:** https://www.brandcrowd.com/maker/tag/ai
- **Background:** Use Unsplash.com for high-quality photos

## Example File Structure

```
frontend/assets/
├── favicon.ico         # Browser/app favicon
├── icon.png           # Desktop app icon (512x512)
├── logo.png           # App logo
├── background.jpg     # Optional background image
└── README.md
```
